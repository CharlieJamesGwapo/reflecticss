  Solution:
    1. Check browser console (F12)
    2. Check terminal for error messages

  Solution:

    3. Should see login page

  Solution:
    1. Verify backend is running on port 5000

    3. Verify DATABASE_URL in backend/.env
    4. Restart backend if needed
