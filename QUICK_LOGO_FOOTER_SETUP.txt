================================================================================
  TROUBLESHOOTING
================================================================================

ISSUE: Logo not showing in navbar
FIX: 
  1. Check logo.png exists in frontend/public/
  2. Check file name is exactly "logo.png"
  3. Check src="/logo.png" in Navbar.js
  4. Restart frontend server

ISSUE: Footer not showing
FIX:
  1. Check Footer.js exists in frontend/src/components/
  2. Check Footer imported in App.js
  3. Check Footer component added to JSX
  4. Restart frontend server

ISSUE: Google Maps not loading
FIX:
  1. Check internet connection
  2. Check iframe src URL is correct
  3. Check no CORS errors in console
  4. Try refreshing page

ISSUE: Footer not responsive
FIX:
  1. Check Tailwind CSS classes are correct
  2. Check grid-cols-1 sm:grid-cols-2 lg:grid-cols-4
  3. Check responsive padding classes
  4. Restart frontend server
