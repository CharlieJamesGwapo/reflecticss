================================================================================
  TROUBLESHOOTING
================================================================================

ISSUE: Logo not showing in navbar
SOLUTION:
  1. Check logo.png exists in frontend/public/
  2. Check file name is exactly "logo.png" (lowercase)
  3. Check src="/logo.png" in Navbar.js (line 93)
  4. Restart frontend server
  5. Clear browser cache (Ctrl+Shift+Delete)
  6. Refresh page (Ctrl+R)

ISSUE: Footer not showing
SOLUTION:
  1. Check Footer.js exists in frontend/src/components/
  2. Check Footer imported in App.js (line 4)
  3. Check <Footer /> in App.js (line 98)
  4. Restart frontend server
  5. Refresh page

ISSUE: Google Maps not loading
SOLUTION:
  1. Check internet connection
  2. Check browser console for errors (F12)
  3. Check iframe src URL is correct
  4. Try refreshing page
  5. Check no CORS errors

ISSUE: Footer not responsive
SOLUTION:
  1. Check Tailwind CSS is working
  2. Check grid-cols-1 sm:grid-cols-2 lg:grid-cols-4
  3. Open DevTools (F12)
  4. Toggle device toolbar (Ctrl+Shift+M)
  5. Check responsive design
