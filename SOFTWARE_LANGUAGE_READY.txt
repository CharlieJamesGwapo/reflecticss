  COMPUTER SOFTWARE AND LANGUAGE (22 NEW TERMS)
================================================================================

1. COMPUTER SOFTWARE
   Definition: A set of programs that tells a computer how to perform tasks.
               It includes system software, which runs the computer, and
               application software, which helps users complete specific tasks.

2. SYSTEM SOFTWARE
   Definition: A type of computer program that manages and controls the
               computer hardware, allowing the system and application software
               to run.

3. APPLICATION SOFTWARE
   Definition: A program that helps users perform specific tasks on a computer
               such as writing a letter. It is designed for practical use by end
               users rather than for running the computer itself.

4. WORD PROCESSING
   Definition: An application software that allows users to create, edit,
               format, and print text documents. It is used for writing letters,
               reports, and other written materials.

5. DESKTOP PUBLISHING
   Definition: An application software that enables users to design and produce
               professional-looking publications. It allows combining text and
               images with precise layout control.

6. DATA HANDLING
   Definition: An application software that manages, organizes, and analyzes
               data stored in databases. It is used for creating, editing,
               sorting, and searching records, and displaying information in
               charts or tables.

7. SPREADSHEET MODELLING
   Definition: An application software that allows users to perform
               calculations, create financial models, or run simulations using
               rows, columns, and formulas. It is used for producing invoices,
               cost plans, and other data-driven models.

8. GRAPHICS PACKAGE
   Definition: An application software that enables users to create, edit, and
               manipulate images and drawings. It is used for digital painting,
               photo editing, and graphic design.

9. COMPUTER-AIDED DESIGN (CAD)
   Definition: An application software that helps users design products, floor
               plans, or 3D models accurately. It is used for engineering,
               architecture, and technical drawing.

10. PRESENTATION SOFTWARE
    Definition: An application software that allows users to create multimedia
                presentations using text, images, video, and sound. It is used
                for lectures, demonstrations, and business presentations.

11. MUSIC SOFTWARE
    Definition: An application software that allows users to compose, record,
                edit, and play musical performances. It is used for creating
                digital music and audio production.

12. PACKAGED SOFTWARE
    Definition: A form of application software that includes multiple programs
                bundled together, used to perform similar tasks efficiently and
                often more cost-effectively than buying programs individually.

13. CUSTOM SOFTWARE
    Definition: A form of application software designed specifically for an
                individual or organization to perform tasks tailored to their
                unique needs.

14. WEB APPLICATION
    Definition: A form of application software that runs through an internet
                browser, allowing user interaction and access to features
                without needing local installation.

15. OPEN-SOURCE SOFTWARE
    Definition: A form of application software whose source code is publicly
                available, allowing anyone to study, modify, and distribute it,
                promoting collaboration and transparency.

16. SHAREWARE SOFTWARE
    Definition: A form of application software initially provided for free,
                allowing users to try and share the program before purchasing
                the full version.

17. FREEWARE SOFTWARE
    Definition: A form of application software that is available at no cost to
                the user, though usually with restrictions on modification,
                redistribution, or reverse engineering.

18. PUBLIC DOMAIN SOFTWARE
    Definition: A form of application software that has no ownership
                restrictions, allowing anyone to freely use, modify, distribute,
                or sell it.

19. PROGRAMMING LANGUAGES
    Definition: Languages that are used to write instructions that tell a
                computer how to perform tasks, forming the foundation for all
                software development.

20. LOW-LEVEL LANGUAGES
    Definition: A type of programming language that interacts closely with the
                computer's hardware, used to create programs that run
                efficiently and quickly.

21. HIGH-LEVEL LANGUAGES
    Definition: A type of programming language designed to be easy for humans
                to read, write, and debug, used to create programs that can run
                on different computers.
