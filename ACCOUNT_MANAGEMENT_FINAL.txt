================================================================================
  API ENDPOINTS
================================================================================

AUTHENTICATION:
  POST /api/auth/register
    - Register with optional photo
    - Returns: { user, token }

  POST /api/auth/login
    - Login with email/password
    - Returns: { user, token }

  GET /api/auth/verify
    - Verify token
    - Returns: { user }

USERS:
  GET /api/users/stats
    - Get user statistics
    - Returns: { lessonsCompleted, quizzesAttempted, averageScore, streakDays }

  PUT /api/users/profile
    - Update profile with optional photo
    - Returns: { user }

  PUT /api/users/change-password
    - Change password
    - Returns: { message }

NOTIFICATIONS:
  GET /api/notifications
    - Get notifications
    - Returns: { notifications, unreadCount }

  PUT /api/notifications/:id/read
    - Mark notification as read
    - Returns: { notification }

================================================================================
  TROUBLESHOOTING
================================================================================

ISSUE: "Failed to update profile"
  SOLUTION:
  1. Check backend is running (npm run dev)
  2. Check .env has CLOUDINARY credentials
  3. Check browser console (F12) for errors
  4. Check backend logs
  5. Verify token is valid (login again)

ISSUE: Photo not uploading
  SOLUTION:
  1. Check file is image (jpg, png, gif)
  2. Check file size < 5MB
  3. Check Cloudinary credentials in .env
  4. Check network connection
  5. Check browser console

ISSUE: "Current password is incorrect"
  SOLUTION:
  1. Verify password is correct
  2. Check caps lock
  3. Login again
  4. Reset password if forgotten

ISSUE: Form not submitting
  SOLUTION:
  1. Check all fields are filled
  2. Check email format is valid
  3. Check passwords match
  4. Check backend is running
  5. Check browser console

ISSUE: "Cannot find module 'cloudinary'"
  SOLUTION:
  cd backend
  npm install cloudinary multer multer-storage-cloudinary
  npm run dev
