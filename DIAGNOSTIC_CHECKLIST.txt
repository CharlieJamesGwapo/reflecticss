================================================================================
  COMMON ERRORS & SOLUTIONS
================================================================================

ERROR: "Failed to fetch"
  CAUSE: Backend not running
  FIX: cd backend && npm run dev

ERROR: "Cannot find module 'cloudinary'"
  CAUSE: Packages not installed
  FIX: cd backend && npm install cloudinary multer multer-storage-cloudinary

ERROR: "Port 5000 already in use"
  CAUSE: Another process using port 5000
  FIX: Kill the process or use different port

ERROR: "relation 'notifications' does not exist"
  CAUSE: Database table not created
  FIX: Run SQL to create table (see CHECK 6)

ERROR: "Current password is incorrect"
  CAUSE: Wrong password entered
  FIX: Verify password is correct

ERROR: "CORS error"
  CAUSE: Frontend and backend not communicating
  FIX: Check API URL in frontend/.env

ERROR: "Invalid token"
  CAUSE: Token expired or invalid
  FIX: Login again

================================================================================
  BACKEND ERRORS
================================================================================

If backend shows errors, check:

1. Node.js installed?
   node --version

2. Packages installed?
   cd backend
   npm install

3. .env file correct?
   cat .env
   Should have DATABASE_URL, JWT_SECRET, CLOUDINARY credentials

4. Database connected?
   Check DATABASE_URL in .env

5. Port 5000 free?
   netstat -ano | findstr :5000

6. Restart backend
   npm run dev
