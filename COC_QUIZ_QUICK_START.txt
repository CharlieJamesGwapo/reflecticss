================================================================================
  TROUBLESHOOTING
================================================================================

ISSUE: No answer choices showing
  Solution:
    1. Hard refresh: Ctrl+Shift+R
    2. Check browser console (F12)
    3. Verify backend running
    4. Restart frontend: npm start

ISSUE: Submit button not working
  Solution:
    1. Make sure answer is selected
    2. Check browser console
    3. Verify backend running
    4. Restart frontend

ISSUE: Progress not updating
  Solution:
    1. Hard refresh page
    2. Check browser console
    3. Verify backend running
    4. Restart frontend

ISSUE: COC 2/3 not showing
  Solution:
    1. Verify routes added
    2. Check App.js imports
    3. Restart frontend
    4. Hard refresh browser
